# iSpend

## Project Description

iSpend is a simple web application that helps users track their income and expenses. Users can add transactions, view transaction history, delete transactions, and monitor their total balance.

## Author

Augustus Mocha

## Setup Instructions

1. Download or clone the project files.
2. Open the project folder in VS Code.
3. Open `index.html` in a browser.
4. Use the navigation links to access other pages.

## BDD (Behavior Driven Development)

| Behaviour                  | Input                               | Output                                  |
| -------------------------- | ----------------------------------- | --------------------------------------- |
| User adds a transaction    | Transaction details entered in form | Transaction is saved and displayed      |
| User leaves fields empty   | Empty form fields                   | Alert message displayed                 |
| User deletes a transaction | Click delete button                 | Transaction is removed from table       |
| User visits homepage       | Existing saved transactions         | Income, expenses, and balance displayed |

## Technologies Used

### HTML

Used to create the structure of the application including forms, tables, navigation bars, and page sections.

### CSS

Used to style the application, create responsive layouts, improve spacing, colors, and overall user interface design.

### JavaScript

Used to handle form submissions, validate inputs, manipulate the DOM, display transactions dynamically, calculate totals, and delete transactions.

### localStorage

Used to store transaction data in the browser so that saved transactions remain available even after refreshing the page.

### Git and GitHub

Used for version control, tracking project progress through commits, and hosting the project repository online.
